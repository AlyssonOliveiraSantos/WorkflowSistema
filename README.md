Projeto – WorkflowSistema
Repositório

Pré-requisitos

Docker instalado

Docker Compose instalado

Instalação e Execução

Clone o repositório:

git clone https://github.com/AlyssonOliveiraSantos/WorkflowSistema


Entre na pasta:

cd WorkflowSistema


Construa as imagens:

docker compose build


Suba os containers:

docker compose up

Endpoints

Backend (API): http://localhost:5000

Frontend: http://localhost:4200
